
# 🚀 Developer Productivity Dashboard

## 📌 Overview

This project is a simple developer productivity dashboard that helps track key engineering metrics and provides insights and suggestions.

---

## 🎯 Problem

Developers can see raw metrics but often struggle to understand:

* What do these metrics mean?
* What should they improve?

---

## 💡 Solution

This dashboard:

* Calculates key metrics
* Shows insights
* Provides actionable suggestions

---

## 📊 Metrics Implemented

* Lead Time for Changes
* Cycle Time
* PR Throughput
* Deployment Frequency
* Bug Rate

---

## 🧠 Features

* Select developer and month
* View performance metrics
* Get insights based on thresholds
* Receive suggestions for improvement

---

## 🛠 Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Node.js + Express

---

## 📁 Project Structure

```
devProductivity-dashboardTool/

│   ├── package.json
│
├── backend/
│   ├── data/       # Jira, PR, Deployment, Bug data
│   ├── server.js   # API logic
│   ├── package.json
│
├── README.md
```


## ▶️ How to Run Locally

### Backend

cd backend
npm install
npm run server

### Frontend

cd frontend
npm install
npm run dev
